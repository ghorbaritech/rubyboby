# Ruby Boby: The AI Companion for Every Child

Ruby Boby is a cutting-edge SaaS platform designed to provide a safe, engaging, and educational conversational experience for children aged 2–10. It features customizable AI personalities that act as friends, teachers, or family members, appearing as interactive human figures on screen to talk and play with children in their native languages.

## 🌟 Vision
To bridge the gap between technology and child development by creating an AI companion that doesn't just answer questions, but nurtures curiosity, emotional intelligence, and linguistic skills in a safe, parent-controlled environment.

## 🚀 Key Features

### 1. Customizable Personalities
Adult admins (parents/guardians) can create and define multiple personalities:
*   **Default Figures:** Start immediately with **Ruby (Girl)**, **Boby (Boy)**, or **The Teacher**.
*   **Diverse Roles:** Friend, Teacher, Caregiver, Family Member, or Imaginary Friend.
*   **Unique Traits:** Define names, behavior patterns, talking styles, and specific response tendencies.
*   **Visual Identity:** Upload images for each personality to create a unique visual presence.

### 2. Interactive Talking Partner
*   **Human-Like Interaction:** The AI appears as a human figure on screen.
*   **Lip-Syncing:** Real-time animation that matches the AI's speech for a realistic "talking" experience.
*   **Multilingual Support:** Supports major languages including **Bangla, English, Hindi**, and others.

### 3. Intelligent Conversation
*   **Age-Appropriate Responses:** Merit-based answers tailored for the 2–10 age bracket.
*   **Proactive Engagement:** The AI asks repeat questions and continues conversations to keep the child engaged.
*   **Memory:** Remembers previous interactions to build a "friendship" over time.

### 4. Admin Dashboard (SaaS)
*   **Character Management:** Create, edit, and delete personalities.
*   **Safety Monitoring:** Review conversation summaries and set limits.
*   **Language Settings:** Configure preferred languages for the household.

## 🛡️ Safety & Compliance (The Core Pillar)
Safety is at the heart of Ruby Boby. The platform is built with "Safety by Design" principles:
*   **COPPA & GDPR-K Compliance:** Strict adherence to child data protection laws.
*   **Input/Output Filtering:** Secondary AI layers to block inappropriate content (violence, grooming, etc.).
*   **Parental Consent:** Mandatory adult verification for account creation.
*   **Deterministic Guardrails:** Hybrid model combining "safe" pre-scripted nodes with controlled generative AI.

## 🛠️ Technical Stack (Proposed)
*   **Frontend:** Next.js (Web App) / Flutter (Mobile App).
*   **AI Brain:** OpenAI GPT-4o / Claude 3.5 Sonnet (Personality-tuned).
*   **Voice (TTS):** ElevenLabs / OpenAI TTS (Multilingual).
*   **Visual Animation:** HeyGen API / D-ID / Audio2Face (Lip-syncing avatars).
*   **Backend:** Node.js / Python (FastAPI) with Vector Database (for memory).
